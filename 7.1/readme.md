# Laravel Test Runner - PHP 7.1

Docker Container with PHP 7.1 and extensions to be compatible with most Laravel applications. Also installed on this container we have Composer and NodeJS/NPM/Yarn.

## Building and pushing the image

**Build**:

```
docker build --pull -t kirschbaumdevelopment/laravel-test-runner .
```

**Tag**:

```
docker tag kirschbaumdevelopment/laravel-test-runner:latest kirschbaumdevelopment/laravel-test-runner:7.1
```

**Push**:

```
docker push kirschbaumdevelopment/laravel-test-runner:7.1
```
